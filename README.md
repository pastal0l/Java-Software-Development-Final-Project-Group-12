# Java Pseudo-3D Raycaster

A minimal Java project using `JFrame` and `JPanel` to demonstrate:

- 60 FPS game loop using `Timer`
- Simple 2D integer map
- WASD movement and left/right rotation
- Basic pseudo-3D raycasting rendering

## Run

From the project root:

```powershell
javac src\Main.java src\GamePanel.java
java -cp src Main
```

## Controls

- W: move forward
- S: move backward
- A: strafe left
- D: strafe right
- Left / Right arrows: rotate


##map
- generateRandomMap() to make the map random
